Vimwiki使用入门

提醒：本教程暂只适用于Vimwiki-1-1-1，新版有少许语法进行了变更，并不向后兼容，请阅者注意
使用Vimwiki有近20天了，感觉从未有过的痛快。之前我也用过Z-Blog、Wordpress、emlog等带数据库的博客系统，但总是不那么畅快，一则数据容易丢失，二则不便于迁移，之后很长一段时间我就没再写博客了。发现Vimwiki最是适合我，纯HTML代码想往哪放就往哪放，再也不怕数据丢失了，HTML相比SQL语句而言，想出问题都难

感谢Vimwiki的开发者提供这么好用的东东，既然没能帮忙贡献代码，就只好帮忙推广推广了，不然对不起开源社区啊

我觉得个人博客不要搞什么数据库驱动的好，一个人就算一辈子也写不了多少文字，自己手动归类整理最好不过了，用不着数据库。此外，我不赞成使用某些助理软件，使用这些软件就相当于你的数据被其绑定了，以后想更换工具基本上没得可能，这也是我试用过CintaNotes后迅速将其放弃的原因。

好了，不聊这些杂七杂八的了，下面进入正题

必备条件

不是谁都可以用Vimwiki的，你首先得具备以下两个条件：

  1. 会使用Vim或Gvim（这两者本质是一样的，只不过后者加了个漂亮的“壳”而已）
  2. 会使用最基本的HTML标签

老实说Vimwiki的用户应该主要就是程序员，基本上只有程序员才会去用Vim这样“古怪”的编辑器，而作为程序员大多都是有网页设计基础的，所以本文针对的就是会使用Vim的程序员

如果不会Vim但想学习怎么使用的，请Google：“Vim 教程”，你会发现资料多了去了（本文末尾也提供了教程供下载），只要肯花上一个星期你就会是Vim好手

必备工具

  • Vim 7.x，最好是7.3或以上
  • vimwiki-xxxx.vba（这儿不提供下载，该项目网址是：http://code.google.com/p/vimwiki/）

安装指南

首先确保你安装好了Vim，其次确保你已经下载好了vimwiki-xxxx.vba文件（当前最新为vimwiki-1-1-1.vba）

  1. 使用Vim打开该vimwiki-xxxx.vba文件
  2. 在Normal模式下执行命令：:so %，不多久就会自动安装完成
  3. 在Vim配置文件中添加以下语句： 
    " 打开语法高亮
syntax enable
syntax on

" 打开对文件类型插件的支持
filetype indent on
filetype plugin on
filetype plugin indent on

" ######### VimWiki 写作助手 ######### "

" 使用鼠标映射
let g:vimwiki_use_mouse = 1

" 不要将驼峰式词组作为 Wiki 词条
let g:vimwiki_camel_case = 0

let g:vimwiki_list = [{
\ 'path': 'D:/mysite/wiki',
\ 'path_html': 'D:/mysite/html/',
\ 'auto_export': 1,
}]

下面对配置文件做下解说

  • path：指定你存放wiki源文件的目录
  • path_html：指定编译得到的HTML文件放到哪个目录
  • auto_export：每当保存wiki源文件时自动编译，关闭设为0

现在再次打开 Vim，直接敲入 \ww，这样就进入了你的wiki首页

基本语法

别见VimWiki小巧玲珑的，规则倒着实不少，下面是我用的最多的几个标签：

  • == 二级标题 ==：二级标题，我一般用来标识文章标题（如果行首空一个空格则会居中，适用于所有标题标签）
  • === 三级标题 ===：三级标题，我一般用来标识文章子标题
  • ====== 六级标题 ====：六级标签，我用来标记时间（标题标签共6个，对应于等于号的个数，六级标签字体最小，一级标签字体最大，本博客首页那些粉红色标题就是用的一级标签）
  • [[链接地址|链接标题]]：这主要用来链接本地文件，如本页首部的[首页] - [Linux] - [Vim]，其对应写法为[[../../../index|首页]] - [[../index|Linux]] - [[index|Vim]]
  • [链接地址 链接标题]：这主要用来贴链接，如我想贴个Google首页的链接可以这样写 [http://www.google.com Google]，不过这个标签我基本上没怎么用过，因为都没几个外部链接
  • [[转向地址|图片地址]]：本博简介页面的头像是这样写的[[index.html|images/ruchee.jpg]]，这样就会显示对应的图片，而点击图片就会转到首页。如果不需要转向，直接写[[图片地址]]即可
  • ----：四条小横线相当于< hr >标签，会出现一条本博客到处可见的横线
  • {{ { 程序代码或不需要编译的内容 } }}：三个大括号用来书写程序代码或其他不想被编译的内容。注意，为了防止这些演示标签生效，我加上了空格，实际使用时不要留空格
  • 两个反斜点（这儿不便写演示代码，反斜点一般位于数字键1的左边）：相当于< code >与< /code >标签，显示效果就是本页随处可见的绿色荧光字

以上就是我用的最多的Vimwiki标签，更多帮助请参考本文末尾下载包附带的手册

我不太会写更深入的教程，更详尽的介绍请参考丘迟的维基

常用命令

  1. :VimWiki2HTML：将当前页面编译成HTML文件
  2. :VimWikiAll2HTML：将本wiki所有源文件都编译成HTML文件（如果文件很多，将花费较长时间）
  3. :VimWikiGoTo：转到指定的wiki页面
  4. :VimWikiIndex：转到wiki首页
  5. :VimWikiSearch /Ruchee/：搜索指定关键字，:cnext跳到下一个，:cprevious跳到上一个
  6. :VimWikiTable 3 2：插入一个2行3列的表格。注意，列数在前，行数在后
  7. :VimWikiDeleteLink：删除当前页面

更多命令使用方法请参考Vimwiki自带的帮助文件，使用命令：:help VimWiki

更多配置

  1. html_header：指定编译时使用的头部模板
  2. html_footer：指定编译时使用的尾部模板
  3. nested_syntaxes：指定在使用三个大括号书写程序代码时需要语法高亮的编程语言
  4. let g:vimwiki_valid_html_tags=：声明可以在 wiki源文件中书写的HTML标签

更多帮助请参考Vimwiki文档（借助Google可以很轻易地找到其文档的中文翻译稿，需要的自己动手找吧，自己动手，丰衣足食啊）
